# CustomTabLayout
自定义TabLayout V1.0.0
Step 1.Add it in your root build.gradle at the end of repositories:
```java
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
Step 2. Add the dependency
```java
dependencies {
	        implementation 'com.github.ZKXSparke:CustomTabLayout:v1.0.0'
	}
```
